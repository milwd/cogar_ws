Control Wheel

control_wheel.py

Action-level base controller for following a coarse navigation path

ControlMovementServer exposes the :ros:`tiago1/MovementControlAction` namespace. Each goal carries a complete :rosmsg:`nav_msgs/Path`; the server steers the robot towards the first waypoint only with a proportional controller and then reports succeeded. Replace the placeholder loop with a full trajectory tracker when integrating real localisation / odometry.

Control logic

Read first waypoint → compute planar error → send velocity

\(v_x = k_p \cdot \Delta x,\; v_y = k_p \cdot \Delta y\) with kₚ = 0.01. The controller runs once per goal (fire-and-forget stub). Real deployment should iterate over every pose, add angular control and stop conditions.
